# AWZ: QuickMail

### [Установка модуля](https://github.com/azahalski/awz.quickmail/tree/main/docs/install.md)

<!-- desc-start -->

Модуль для немедленной отправки почтовых событий в 1С-Битрикс.

## Основная функциональность

Модуль `awz.quickmail` позволяет отправлять отмеченные почтовые шаблоны немедленно через `Bitrix\Main\Mail\Event::sendImmediate()` сразу после записи в таблицу `b_event`. 

### Ключевые возможности:

- **Немедленная отправка**: Письма отправляются сразу же, минуя очередь стандартного агента
- **Удаление из очереди**: Опциональное удаление отправленных писем из очереди `b_event` для предотвращения дублирования
- **Блокировка отправки**: Возможность полностью заблокировать отправку писем через стандартный механизм
- **Удобный интерфейс**: Настройка как на уровне списка шаблонов, так и при редактировании конкретного шаблона

## Как это работает

1. При добавлении письма в очередь (`OnAfterEventAdd`) модуль проверяет, отмечен ли шаблон для немедленной отправки
2. Если да, письмо отправляется немедленно через `sendImmediate()`
3. Если включена опция удаления, письмо удаляется из очереди после успешной отправки
4. При попытке отправки через обычный механизм (`OnBeforeEventSend`) модуль блокирует отправку
